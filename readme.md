## Prerequisites

- Chromium browser

## Install

- Clone this repository to your pc
- Unzip it
- Go to chrome://extensions -> Load unpacked & select project folder

## Usage

- Click an extension on top-right on your browser, then select the font you want

## Add your font to extension

- Open js/fonts.js
- Add your font name in `fonts` array
- Ex: You want to add you font named `"Beautiful Font"`, just add that name in `fonts` array ğŸ‘‰ fonts = ['Cambria', 'Helvetica Neue', 'Roboto', 'Beautiful Font'];
